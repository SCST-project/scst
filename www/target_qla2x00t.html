Target driver qla2x00t for QLogic FC adapters

This is a target driver for QLogic qla2xxx (22xx++) Fibre Channel adapters. It is stable and well tested.

This driver starting from version 3.1 supports 16G Hilda QLogic chip based adapters (post-Hilda QLogic chips not supported) and has many other important improvements. This driver should also support FCoE, but that has never been verified. It has passed intensive internal SanDisk tests. It is stable and production ready. This is the recommended driver to use in production at the moment.

A newer and experimental QLogic target driver is available in the qla2x00t-32gbit directory. That driver supports all adapters supported by the qla2x00t driver and also several newer adapters. The existing qla2x00t driver will be replaced by the qla2x00t-32gbit driver once it is stable enough.
